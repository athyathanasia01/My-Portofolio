Projek Skripsi | Optimalisasi Penerapan Algoritma Clustering dalam Meningkatkan Performa Aturan Asosiasi FP-Growth untuk Rekomendasi Stok Penjualan Radio Komunikasi

Projek skripsi menentukan rekomendasi stok menggunakan perbandingan algoritma clustering yaitu K-Means, HDBSCAN, dan fuzzy c-means, yang masing- masing memiliki karakteristiknya sendiri-sendiri. Hasil algoritma terbaik kemudian dipilih berdasarkan nilai evaluasi Silhouette, DBI, dan Calinski-Harbasz Score, untuk memfilter data transaksi penjualan dari setiap cluster terbentuk dan dibandingkan dengan keseluruhan data transaksi. Kemudian filter dan keseluruhan data transaksi diproses dengan aturan asosiasi FP-Growth dan dibandingkan hasilnya. Didapatkan salah satu cluster memiliki nilai evaluasi support tertinggi sebesar 10% dengan produk yang sama juga muncul pada penerapan FP-Growth untuk data transaksi keseluruhan dengan evaluasi support tertinggi juga.

Projek Mata Kuliah SPK | Powery Web App

Projek aplikasi website mata kuliah Sistem Pendukung Keputusan. Ini merupakan project software berbasis website yang dibangun di atas framework PHP sebagai konektivitas dengan local database (MySQL). Database digunakan sebagai penampung hasil dari perhitungan Analytical Hierarchical Process (AHP). Input penilaian berasal dari User dan penilaian dilakukan otomatis menggunakan model AHP dari Python. Hasil dari penilaian ditampilkan secara berurutan sebagai rekomendasi personal user. Projek ini berhasil maju ke prosiding ICONICSS 2024.
